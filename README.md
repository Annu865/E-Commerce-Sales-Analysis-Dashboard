\# 🛒 E-Commerce Sales Analysis Dashboard



\## Project Overview



This project analyzes E-Commerce sales data using SQL and Power BI. The project focuses on revenue analysis, product performance, customer analysis, payment methods, and sales trends.



\---



\## Tools Used



\- SQL (PostgreSQL)

\- Power BI



\---



\## Dataset Tables



\- Customers Dataset

\- Orders Dataset

\- Order Items Dataset

\- Order Payments Dataset

\- Products Dataset



\---



\## SQL Analysis



\### Total Revenue

Calculated total revenue generated from all orders.



\### Revenue by State

Analyzed revenue contribution by state.



\### Revenue by Payment Type

Compared revenue generated by different payment methods.



\### Top Products by Revenue

Identified top revenue-generating products.



\### Monthly Revenue Trend

Analyzed monthly revenue performance.



\### Top Revenue Customers (CTE)

Used Common Table Expressions (CTE) to find top customers by revenue.



\### Revenue Ranking (Window Function)

Used Window Functions to rank payment methods based on revenue.

\## Key Findings



\- Credit Card is the highest revenue-generating payment method.

\- SP is the highest revenue-generating state.

\- A few products contribute most of the revenue.

\- Revenue varies across months.
